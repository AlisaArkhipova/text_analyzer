# Text Analyzer

## О проекте
Text Analyzer — это консольное приложение на Python для базового анализа текста.
Программа помогает быстро оценить объём текста и частотность слов.
Подходит для учебных задач по Python и работе с Git/GitHub.

## Функциональность
- Ввод текста вручную или чтение из файла
- Подсчёт символов (с пробелами и без)
- Подсчёт слов и предложений
- Топ-N самых частых слов

## Требования
- Python 3.10+
- Внешние библиотеки не требуются
